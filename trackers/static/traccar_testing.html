Traccar testing

  1. Install "Traccar Client" app.

    Click here for Android

    Click here for iOS

  2. Open app. Set the following settings:
    • Server Url: http://trackrace.tk:5055 (Important. You must change this!)
    • Frequency: Choose as you desire. Recommended: 300 = 5min
    • Distance/Angle:   (Leave blank) (Default)
  3. Make sure your gps is not disabled on your phone.
  4. Start service
  5. Note Device identifier
  6. Check the tracker is working in a browser. Go to https://trackrace.tk/traccar/[DEVICE IDENTIFIER]
  7. Go for ride
  8. Stop service
  9. Let Gary know how it went :-)

Notes

  • The tracking page will show tracking for the last 7 days.
  • Any one who knows your device identifier will be able view the tracking data. It will be difficult for someone to guess this, but not impossible. So make sure the traccar service is off if you are going some where that don't want other people to know about. If you want me to delete some traking data, just send me a message.
