Traccar testing

  1. Install "Traccar Client" app.

    Click here for Android

    Click here for iOS

  2. Open app. Set the following settings:
    • Server address: trackrace.tk (Must change this!)
    • Server port: 5055 (Default value)
    • Encryption: disabled (Default value)
    • Frequency: Choose as you desire. Recommended: 300 = 5min
    • Distance/Angle: Choose as you desire. Recommended:   (Leave blank) (Default) which will turn this feature off. Turning this on will keep the GPS on all the time, which will use more battery.
    • Location Provider: GPS Provider (Default)
  3. Start service
  4. Note Device identifier
  5. View tracker in browser. Go to https://trackrace.tk/traccar/[DEVICE IDENTIFIER] This page will show tracking for the last 7 days.
  6. Go for ride
  7. Stop service
  8. Let Gary know how it went :-)
