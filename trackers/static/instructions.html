Usage instructions

Install Client - Android

  1. Install TrackRace Client app.

Install Client - Apple iOS

  1. Install Traccar Client app.
  2. Open app. Set the following settings:
    • Server Url: http://trackrace.tk:5055
    • Accuracy: High

Usage

  1. Open app.
  2. Start service
  3. Note Device identifier
  4. Go to https://trackrace.tk/traccar/[DEVICE IDENTIFIER] to view tracker.
  5. Stop service
