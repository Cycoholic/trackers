Trackrace.tk Personal Use Instructions

Note: This service is provided without warranty. My focus is on providing event tracking, and this whole thing is done in my spare time. You are welcome to use it, but if it stops working, I make no promises of how long I will take to fix it.

App Setup

  • Android Phones
    1. Install TrackRace Client app.
    2. Open app.
    3. Note (or change) the Device identifier.
  • Apple Phones
    1. Install Traccar Client app.
    2. Open app.
    3. Set the following settings:
      • Server Url: http://trackrace.tk:5055
      • Accuracy: High
    4. Note (or change) the Device identifier.

Use

  1. Start the service in the app.
  2. View tracking page in a browser
    Go to https://trackrace.tk/traccar/DEVICEID or enter your device id below:
  3. Stop the service in the app.
